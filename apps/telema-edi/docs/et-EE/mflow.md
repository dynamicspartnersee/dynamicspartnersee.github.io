# Telema MatchFlow

Lugege lähemalt MatchFlow kohta Telema kodulehelt: [https://eflow.live/et/mflow/](https://eflow.live/et/mflow/)

MatchFlow abil saate ostutellimuse ja tarne vastendada saabunud arvega. Selleks tuleb ostutellimus ja tarne saata MatchFlow-sse ning saabunud arve seal vastendada ja kontrollida, misjärel on see valmis vastuvõtmiseks ERP-i.

## Hankija seadistamine
MatchFlow kasutuselevõtmiseks soovitud hankija puhul, seadista hankija kaart alljärgnevalt:

| Väli | Väärtus | Selgitus |
| - | - | - |
| **Telema mFlow** | 3-poolne vastendus | MatchFlow-sse saadetakse nii tellimus kui tarne. |
| **Väljasta e-tellimus** | Jah või Ei | Jah, kui soovid tellimust saata ka hankijale, ei, kui ainult MatchFlow-sse. |
| **Väljasta e-tarne** | Jah või Ei | Jah, kui soovid tellimust saata ka hankijale, ei, kui ainult MatchFlow-sse. |

## Hankijate saatmine
Soovi korral saate MatchFlow keskkonda saata hankijate andmed.  

Partnerite faili (e-dokument, mille liik on 'partners') loomiseks, avage **Väljaminevad e-dokumendid (EDI)** ning käivitage tegevus:
Toimigud -> Uus -> Loo dokument BC-st -> **Loo partnerid**

## Ostutellimuse saatmine
Ostutellimuse saatmiseks MatchFlow-sse avage ostutellimus ning faktikastis **Telema e-dokument** käivitage tegevus **Väljasta e-tellimus**.

Juhul kui hankija kaardil on märgitud nii **Telema mFlow** kui ka **Väljasta e-tellimus**, siis väljastatakse kaks tellimust, üks MatchFlow ja teine hankija jaoks.
